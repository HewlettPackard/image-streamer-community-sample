ABSee

Image Streamer Artifact Bundle Viewer

ABSee allows you to:

      View contents of artifact bundles without importing them on an Image Streamer appliance
      Open multiple artifact bundles at the same time
      View plan scripts with syntax highlighting
      Compare artifacts from same or different bundles
      Compare entire artifact bundles
      Export artifact bundles in human-readable YAML format
      Cross-platform. Available for Windows, Mac OS and Linux
